# АРР 001: Tauri вместо Electron для Windows

**Дата**: 2024  
**Статус**: Принято

## Контекст

Приложение браузера для Windows нуждается в оболочечном фреймворке, который встраивает движок веб-рендеринга и предоставляет доступ к нативным API ОС (файловая система, уведомления, системный трей, списки переходов).

## Рассмотренные варианты

### Вариант А: Electron
- Комплектует полную среду Chromium + Node.js
- Зрелая экосистема, широко используется
- Размер бинарника: ~100–150 МБ для минимального приложения
- Значительные накладные расходы по памяти (Chromium + Node)
- Поддерживает расширения Chrome «из коробки»

### Вариант Б: Tauri
- Использует предоставляемый ОС WebView2 (Chromium на Windows)
- Бэкенд на Rust — небольшой, быстрый, безопасный с точки зрения памяти
- Размер бинарника: ~5–15 МБ для минимального приложения
- Нет встроенного браузерного движка — зависит от системного WebView2 (всегда присутствует на Windows 10/11)
- Поддержка расширений: не встроена (потребует кастомной среды WebExtensions)

## Решение

**Tauri** выбран основным фреймворком по следующим причинам:
1. **Размер**: бинарники в 10–15× меньше ускоряют загрузку и запуск.
2. **Безопасность**: Rust-бэкенд сокращает поверхность атаки; модель разрешений Tauri по умолчанию более строгая, чем у Electron.
3. **Производительность**: Node.js не задействован в критическом пути; IPC осуществляется через каналы, а не HTTP.
4. **Доступность WebView2**: поставляется со всеми установками Windows 10 (1803+) / Windows 11 и автоматически обновляется через Windows Update.

### Принятые компромиссы
- Поддержка расширений Chrome недоступна в MVP. Это приемлемо для v0.1; будущий АРР может рассмотреть уровень совместимости с WebExtensions.
- API Tauri меньше, чем у Electron — некоторые функции требуют кастомных Rust-команд.

## Последствия
- `apps/windows/src-tauri/` содержит Rust-бэкенд.
- Фронтенд — стандартное приложение Vite + React, обслуживается локально во время разработки и упаковывается в бинарник в продакшне.
- Поддержка расширений отложена на будущий этап.
